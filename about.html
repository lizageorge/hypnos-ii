About Hypnos

Hypnos is a simple sleep tracker that lets you log when you went to bed each night and when you woke up each morning so that you can see how your sleep patterns change over time. If you’re looking to improve your sleeping habits - get more sleep every night, maintain a more regular sleep schedule, or shift it for a new school or work schedule - this is a great tool that allows you to track your progress.
Hypnos is also my personal project, and a way for me to learn and practice web development. There will be several versions made, each using a different tool or technique that I am currently learning. As this is a work in progress, you might not be currently viewing the most recent version of Hypnos (as of right now, this one isn’t even functional). To track the progress of this project, visit my github page at https://github.com/lizageorge/hypnos-ii
